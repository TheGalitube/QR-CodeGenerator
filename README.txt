# QR Code Generator

Ein einfacher QR-Code-Generator, der es Benutzern ermöglicht, QR-Codes für Text, URLs und WLAN-Zugangsdaten zu erstellen.

## Funktionen

- **QR-Code-Typen**: Wählen Sie zwischen Text, URL und WLAN.
- **Eingabefelder**: Geben Sie die erforderlichen Informationen ein, um den QR-Code zu generieren.
- **Download**: Laden Sie den generierten QR-Code als PNG-Bild herunter.

## Verwendung

1. Wählen Sie den gewünschten QR-Code-Typ aus dem Dropdown-Menü.
2. Geben Sie die erforderlichen Informationen in die Eingabefelder ein.
3. Klicken Sie auf die Schaltfläche "QR Code generieren", um den QR-Code zu erstellen.
4. Klicken Sie auf die Schaltfläche "QR Code herunterladen", um das Bild zu speichern.
